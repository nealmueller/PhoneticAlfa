# Phonetic Alfa

Phonetic Alfa helps you read codes clearly over calls using NATO or LAPD phonetics.

## Tabs
- Drill: quick letter/word drill mode
- Quiz: short repeatable quiz rounds with score history
- Speak: type a code and hear/copy readback
- Settings: remove ads, restore purchase, rate, share, feedback

## Core behavior
- Input is limited to 30 characters
- Live translation as you type
- Supports both NATO and LAPD alphabets across Drill, Quiz, and Speak
- Tap output to copy with haptic feedback
- Speak/Stop playback with `AVSpeechSynthesizer`

## Ads and purchase
- Free version shows a banner ad area above the tab bar
- One-time non-consumable purchase removes ads
- Restore Purchase re-enables ad-free access for the same Apple ID
